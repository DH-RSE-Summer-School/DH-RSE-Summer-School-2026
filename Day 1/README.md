## Digital Humanities at the Command Line

In this session we will introduce command which can be found on any Unix (i.e. Linux and Mac) device, various arguments and flags which these commands can take, and we'll look in particular at piping from one command to another. This session should prepare you for running HPC jobs via the command line.

After the break, there will be a command line group challenge, aimed at consolidating your knowledge and showing how fairly simply commands can build detailed bespoke pipelines.
